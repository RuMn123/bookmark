如何导入浏览器书签

本指南将帮助您将浏览器中的书签导出并导入到导航站中

Chrome浏览器书签导出步骤

  1. 打开Chrome浏览器，点击右上角的"三点"菜单图标
  2. 选择"书签" > "书签管理器"
  3. 在书签管理器页面中，点击右上角的"三点"图标
  4. 选择"导出书签"
  5. 选择保存位置，点击"保存"按钮，默认文件名应为"书签_日期.html"

Firefox浏览器书签导出步骤

  1. 打开Firefox浏览器，点击右上角的"三杠"菜单图标
  2. 选择"书签" > "管理书签"
  3. 在书签库窗口中，点击"导入和备份"按钮
  4. 选择"导出书签为HTML..."
  5. 选择保存位置，点击"保存"按钮

Microsoft Edge浏览器书签导出步骤

  1. 打开Edge浏览器，点击右上角的"..."菜单图标
  2. 选择"收藏夹" > "管理收藏夹"
  3. 点击右上角的"..."图标
  4. 选择"导出收藏夹"
  5. 选择保存位置，点击"保存"按钮

将书签导入到导航站

  1. 回到导航站首页
  2. 点击"导入书签文件"区域的"选择文件"按钮
  3. 选择您刚才导出的HTML书签文件
  4. 等待片刻，您的书签将被解析并显示在页面上
  5. 如果导入成功，您会看到导入成功的提示

或者，您可以直接将书签HTML文件拖放到导航站页面中进行导入。
